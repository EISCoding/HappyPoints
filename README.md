# Happypoints – Upload Ready

Diese Version ist für klassisches PHP-Hosting / Plesk gedacht.

## Upload
Lade **den kompletten Inhalt** dieses ZIP in dein Zielverzeichnis hoch, z. B. `httpdocs/`.

## Voraussetzungen
- PHP 8.1+
- MySQL / MariaDB
- mod_rewrite aktiv
- `.htaccess` erlaubt

## Einrichtung
1. Datenbank anlegen oder `database/init.sql` importieren
2. `config/database.php` anpassen
3. Projekt hochladen
4. Seite aufrufen

## Funktionen
- Registrierung
- Login / Logout
- Dashboard
- Kontostand erhöhen / reduzieren
- Transaktionen
- Partnercode
- Partner verbinden / trennen
- Kontoübersicht
